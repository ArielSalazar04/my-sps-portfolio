My Projects:
  1. Lotería Mexicana (September 2019 - Present)
    The game Lotería Mexicana, translated as "Mexican Lottery", is a game similar to bingo and traditionally played in Mexico. I coded this game as an iOS app and I taught myself how to code in Swift 3.0. While coding this project, I learned about the Main Storyboard, View Controllers, UI Objects, and other fundamental concepts to iOS app developing. I have committed this project to my GitHub.

  2. Puppy Database with Python and MySQL (October 2019 - February 2020)
    This personal project connects Python with MySQL using MySQL Connector. Throughout the dates specified, I participated in a side business, that sold puppies, by storing all the data into a database in MySQL and using Python to make changes in the database and to perform basic statistics.
